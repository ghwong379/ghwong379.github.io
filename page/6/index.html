Elvis Presley

Netflix上出現了這部片的推薦，兩年前去電影院看當時最夯的捍衛戰士2，依稀記得正片放映前有一部預告片就是這一部，兩年後終於補看了這部片。

貓王（Elvis Presley），全名埃爾維斯·亞倫·普雷斯利，是美國歷史上最著名的搖滾歌手之一，他的一生充滿了輝煌與傳奇。

早年生活

貓王於1935年1月8日出生於密西西比州圖珀洛，他有一個孿生兄弟在出生時夭折。他的家庭貧窮，但充滿音樂和宗教的氛圍。貓王從小就展現出對音樂的熱愛，特別是福音音樂、藍調和鄉村音樂。

音樂生涯的開始

1953年，貓王在孟菲斯的太陽唱片公司錄製了他的第一首歌曲。他獨特的音樂風格，結合了節奏藍調和鄉村音樂，很快就吸引了大量的粉絲。

成名與影響

1956年，貓王發行了《Heartbreak Hotel》，這首歌迅速登上排行榜首位，標誌著他的正式成名。他的舞台表演風格獨特，動作充滿性感魅力，引起了巨大的社會反響，被稱為“搖滾之王”。

電影生涯

貓王的影視生涯始於1956年，他在多部電影中擔任主角，如《Love Me Tender》、《Jailhouse Rock》等，這些作品進一步鞏固了他的明星地位。

私人生活

1967年，貓王與普莉希拉·貝里厄結婚，兩人育有一女莉莎·瑪麗。然而，這段婚姻在1973年結束。

晚年與逝世

1970年代，貓王的健康狀況開始惡化，部分原因是長期濫用處方藥物。1977年8月16日，貓王在田納西州孟菲斯的家中去世，享年42歲。

貓王的一生不僅是音樂和文化的一個時代標誌，他對搖滾音樂和整個娛樂界的影響深遠。他的音樂、風格和形象至今仍深深影響著後世的藝術家和樂迷。

膽小鬼遊戲

發表於 2022-07-30 更新於 2024-11-04

近來社會上瀰漫者一股氣息，有人說是膽小鬼遊戲，上網查了一下以為紀念。

膽小鬼賽局（英語：the game of chicken，又譯懦夫賽局、小雞遊戲），又稱鷹鴿賽局（hawk–dove game）或雪堆賽局（snowdrift game），是賽局理論中兩個玩家對抗的模型，一個玩家讓步對於雙方都有好處，而玩家的最佳選擇取決於其對手會做什麼：如果對手讓步，那本方就不應該讓步，但如果對手不讓步，本方就應該讓步。簡而言之就是「不要命的最大」。

膽小鬼賽局收益矩陣
「膽小鬼賽局」這一名稱的由來是一種危險的遊戲，遊戲中兩名車手相對驅車而行。如果兩人拒絕轉彎，任由兩車相撞，最終兩人都會死於車禍；但如果有一方轉彎，而另一方沒有，那麼轉彎的一方會被恥笑為「膽小鬼」（chicken），另一方勝出，因此這賽局模型在英文中稱為「The Game of Chicken」（懦夫遊戲），該術語在政治學和經濟學中普遍使用。「鷹鴿」（hawk–dove）指的是面對同一資源的爭奪中，競爭者可以選擇調和或者對抗的情況，該術語常見於生物學和進化賽局理論。在賽局理論中，兩種叫法並無實質區別，不同的名稱源於不同研究領域的基本原則的平行發展。這種賽局也被用於描述核戰爭中的相互保證毀滅，特別是古巴導彈危機中的邊緣政策。

第五台筆電

隆重推介 Yoga Slim 7。這款超纖薄的機身結合 AMD Ryzen™ 4000 系列行動處理器與 Radeon™ 圖形及長達17.5小時的電池續航力，是專為同級最佳效能和行動力所設計。兼具 Dolby Vision™ 最佳化的 FHD 顯示器，以及 Dolby Atmos®3D 音效，將提供最極致的娛樂體驗，並且將利用智慧省時功能簡化您的日程，當中包括在您離開時自動登出的動態感應功能；電池優化以及在睡眠模式也能運作的語音助手。伴您工作玩樂更明智。

資料來源

報修過程

台北的維修點在106台北市大安區忠孝東路四段270號3樓之2，抽號碼進入等候，由於接近晚間七點才到，當天人不多，前台有三位服務員，仔細檢查機身後簽了兩張單子就回來等候消息。當天即透過簡訊告訴我可自行上網查詢進度，服務滿分。維修人員告之3-5天即會有結果。我記得之前使用x31，保固三年，就在保告到期前螢幕出現不明亮線，到達現場請人員幫我立即處理，當日就取件完成，這也就是為什麼一直喜愛使用IBM的原因。

報修結果

8/4告知將更換主機板、機殼、鍵盤，約9張小朋友離家出走，這就是打翻一杯水的代價，預計明日取機，聽說有個block的問題待解決。

重新安裝

機器取回後，趕忙試了一下，因為bitlock之故，也花了一些時間處理，結果不妙，一直當機，各種重置手段試了又試，也是一直當機，甚至進入BIOS也會當機。最終乾脆重切硬碟，灌了windows 10，當機次數似乎明顯減少，然而找不到麥克風的驅動程式，終究不完美。忍耐一天後，上官網看看還原碟的作法。

這台已超出保固囉。

按照以上步驟下訂後就可以下載囉，準備一個16G的USB，開機時按住F12，選擇USB開機後，一切全自動，真是太厲害囉，安裝完成後，只當過一次，其他一切都好。

後記

結果第一次送修後，換了許多零件，結果仍是維修不完整，運氣不佳患的主機板有問題，只好再次送修，這次連飲料都不想喝，真是很浪費時間的維修送修，總之花錢消災。

Lychee

這是一個相簿管理軟體，非常優秀，之前胡亂摸索但是容易裝壞，猜想是cache的問題。特別是synology內建php版本僅7.2不及7.4，用了很多黑方式，終究是掛掉了。抱著學習的決心，繼續努力測試，目前測試結果步驟如下：

  1. 下載最新版的lychee
  2. 解壓縮至lychee資料夾
  3. WebStation設定虛擬主機，指定至web/lychee/public，並指定php7.4 + Apache2.4
  4. 若有舊的.env，就將它刪除吧
  5. phpmyadmin新建mysql DB
  6. .env須設定mysql，username，password等環境

Flarum

安裝方式

Server Requirements
Before you install Flarum, it’s important to check that your server meets the requirements. To run Flarum, you will need:

  • Apache (with mod_rewrite enabled) or Nginx
  • PHP 7.3+ with the following extensions: curl, dom, gd, json,
  • mbstring, openssl, pdo_mysql, tokenizer, zip
  • MySQL 5.6+ or MariaDB 10.0.5+
  • SSH (command-line) access to run Composer

SANGEAN 三進收音機

發表於 2021-01-29 更新於 2024-11-04 分類於 3C

SANGEAN 三進收音機，工作者的好夥伴

年過半百之後，逐漸希望能將生活方式簡化，特別是不想每天生活在選擇之中，而這些無聊的選擇往往又不是非常必須的。在反省自己的過程中，發現歌曲挑選與撥放常浪費了許多寶貴時間。回想起小時候聽收音機的美好時光，不須多想，只消按下開關，旋鈕，美好的聲音就自小盒子源源不絕傳出。話說從前，小時候對面住著一位叔叔，他的工作場所就在院子。當時並不流行電梯大廈，家家戶戶多為透天厝，小院子也是必須的。那位叔叔在自家院子加工製作喇叭，他將喇叭本體與後方的圓形磁鐵上膠接合在一起，這就是他的工作，至少當時我看到的是如此。在他工作台前方，放有一對沒音箱的喇叭正播送著中廣音樂網的節目，小孩看到這幅畫面非常興奮，心想原來聲音就是這樣出來的，沒有衣服的喇叭也會有聲音，真是非常奇妙。

需要的不多，想要的太多

清楚知道需求之後，決定重拾簡單聆聽的美好時光。三進SANGEAN一直都是收音機的第一品牌，所以當然也在挑選比較之列。期間也曾至三創試聽Geneva藍芽音箱、以及Tivoli的收音機，在試聽比較之後，最終選擇了本土品牌SANGEAN系列的產品，以表達支持產業發展的決心。

PR-D7

首先買了第一台SANGEAN PR-D7，這台的特點是體積不大，喇叭音質也很讚，即便是在大樓中間低樓層的房子，室內收訊能力依然不俗。最特別的地方是電池可使用6顆三號電池，也可將收音機當作充電器使用，這下就能滿足一台收音機可到處帶著走的需求了。第一台SANGEAN PR-D7後來當作孝親機使用，也給我再買一台的機會，有了這次網購的經驗，也增強了信心。

照片來源

WR22

第二台在經過了兩天的思考比較，決定直接挑戰高階的SANGEAN WR22，這款的特點在於是木質音箱且具藍芽及USB的功能，該有的鬧鐘功能貪睡設定等等一應具全。決定當作床頭音響使用，就寢前聽著FM 91.3 Bravo 電台的爵士音樂入睡很是幸福，睡眠品質也提升不少。早晨則是聽著從FM 99.7傳來起床的音樂，也讓人充滿了開心的感覺，音量全開測試也感覺頗有威力。

第三台是因為兒子國三需要準備考試之故，再次買了SANGEAN PR-D7放在書房陪伴他讀書，期待我兒不再花時間開啟電腦打開Youtube 頻道進行選取，而能專心閱讀準備考試，當然我也常常拿到其他地方來聆聽欣賞囉，充分發揮可移動的優勢。


照片來源

DT-125

故事寫到這兒還沒結束， 第四台就是SANGEAN DT-125，買這台的理由是要放在辦公室工作時聆聽。本來在辦公室透過網路進行選取音樂播放，但礙於網路常常不順聲音斷續，影響工作效率，因此決定再買一台。這台嬌小的DT-125只要兩個三號電池就可收聽約半個月的高音質收音機，上起班來果然充滿了精神，收音機開啟後每1.5小時收音機會自動關閉節省用電，算是非常貼心的設計。

附帶說明在選擇適合的收音機時，也比較了WR-11，Mozart藍芽小音箱等產品，這兩款都是木質音箱，復古造型，也是手調選台，最後沒能入手可能這類產品我想這也是主要原因吧。
