Python 擷取網頁上圖片 v2

發表於 2014-12-19 更新於 2024-11-04 分類於 Python

升級版，直接讀取系統產生之學生名單xls檔，唯一要做的事情就是要轉存成新版的xlsx就行

這個版本是用Python 3.4完成，許多函式都是上網查詢試錯，Coding真是有趣又累人的一件事情。心得記錄如下：

from openpyxl import load_workbook

必須先上網安裝openpyxl套件，此套件僅適用於新版的xlsx，舊版的不支援

開啟檔案後，取得工作表名稱，並設定為使用中

for i in range(8, 100):

開啟程式迴圈，一個班最多100人，資料從第8行開始讀取

if str(ws.cell(row=i, column=5).value)\[0:2\] == '10':

比對儲存格前兩個字是否為10，是的話代表那行才是學生資料。這裡用到了ws.cell(row=i, column=5).value這個函數，可以直接取得使用中工作表單中第i列第5欄的資料，並使用str()將之轉為字串，str()[0:2]，並讀取前兩個字

fname = str(ws.cell(row=i, column=1).value)\[6:8\]+str(ws.cell(row=i, column=5).value)+str(ws.cell(row=i, column=6).value)

此行將班別＋學號＋姓名組合而成檔案名稱，等下抓取使用

resource = urllib.request.urlopen("http://path/"+ str(ws.cell(row=i, column=5).value) +".jpg")

開啟檔案抓取模式，urllib.request.urlopen是 Python 3.4的新語法，與2.x版不同

output = open(fname.replace("　","") + ".jpg", "wb")

開啟二進制寫入模式，並用fname.replace(“　“,””)替換檔名中全形空白，因為有學生的名字只有兩個字，系統輸出時會補上全形空白

信耶穌不合科學

發表於 2012-04-21 更新於 2024-11-04 分類於 joke

part1

「信耶穌不合科學。」一個哲學教授上課時說。
他頓了一頓，叫了一個新生站起來，說：「某某同學，你是基督徒嗎？」
學生：「老師，我是。」
教授：「那麼你一定信上帝了？」
學生：「當然。」
教授：「那上帝是不是善的？」
學生：「當然。上帝是善的。」
教授：「是不是上帝是全能的？祂無所不能，對嗎？」
學生：「對。」
教授：「你呢？你是善是惡？」
學生：「聖經說我有罪。」
教授撇撇嘴笑：「哈，聖經。」頓了一頓，說：「如果班上有同學病了，你有能力醫治他，你會醫治他嗎？起碼試一試？」
學生：「會。」
教授：「那麼你便是善的了．．．」
學生：「我不敢這麼說。」
教授：「怎麼不敢？你見別人有難，便去幫助．．．我們大部分人都會這樣，只有上帝帝不幫忙。」
一片沉默。

part2

沒有回答。
老教授同情他了，說：「你無法解釋。對吧？」

學生：「呃．．．，是。」
教授：「魔鬼是善是惡？」
學生：「是惡。」
教授：「那怎麼有魔鬼呢？」學生不知道怎麼回答。
學生：「是．．．是．．．上帝造的。」
教授：「對，魔鬼是上帝造的。對嗎？」
老教授用瘦骨嶙峋的手梳梳稀薄的頭髮，

對傻笑著的全體同學說：「各位同學，相信這學期的哲學課很有興趣。」
回過頭來，又對站著的那同學說：
教授：「世界可有惡的存在？」
學生：「有。」
教授：「世界充滿了惡。對吧？是不是世上所有一切，都是上帝造的？」
學生：「是。」
教授：「那麼惡是誰造的？」
沒有回答。
教授：「世界有不道德的事嗎？有仇恨、醜陋等等一切的惡嗎？」
該學生顯得坐立不安，勉強回答：「有。」
教授：「這些惡是怎麼來的？」
沒有答案。
忽然老教授提高聲調說：「你說，是誰造的？你說啊！誰造的？」
他把臉湊到該學生面前，用輕而穩定的聲音說：「上帝造了這一切的惡。對吧？」
沒有回答。
該學生嘗試也直視教授，但終於垂下了眼皮。
老教授忽然轉過身來，在班前踱來踱去，活像一隻老黑豹。同學們都進入被催眠狀態。

沒有回答。
教授：「世上豈不是充滿了災難？」
停了一下，他又把臉湊到該新生面前，低聲說：「上帝是不是善的？」
沒有答話。
「你信耶穌基督嗎？」他再問。
該學生用顫抖的聲音說：「老師，我信。」
老教授失望地搖了搖頭，說：「根據科學，我們對周圍事物的觀察和了解，是用五官。請問這位同學，你見過耶穌沒有？」
學生：「沒有。老師，我沒見過。」
教授：「那麼，你聽過祂的聲音嗎？」
學生：「我沒有聽過祂的聲音。」
教授：「你摸過耶穌沒有？可有嚐過他？嗅過他？你有沒有用五官來感覺過上帝？」
沒有回答。
「請回答我的問題。」
學生：「老師，我想沒有。」
教授：「你想沒有嗎？還是實在沒有？」
「我沒有用五官來接觸過上帝。」
教授：「可是你仍信上帝？」

學生：「呃．．．是．．．」
老教授陰陰地笑了：「那真需要信心啊！科學上強調的，是求證，實驗，和示範等方法，
根據這些方法，你的上帝是不存在的。對不對？你以為怎樣？你的上帝在哪裡？
學生答不上來。
教授：「請坐下。」
該同學坐下，心中有說不出的沮喪。
這時，另一個同學舉起手來，問：「老師，我可以發言嗎？」老教授笑說：「當然可以。」
學生說：「老師，世界上有沒有熱？」
教授答：「當然有。」
學生說：「那麼，也有冷嗎？」
教授答：「也有冷。」
學生說：「老師，您錯了。冷是不存在的。」
老教授的臉僵住了。課室裡的空氣頓時凝結。

學生說：「老師，」該學生竟又問：「世上有沒有黑暗？」

Intravenous Urogrm靜脈腎盂造影
由於日前腰痛，看了醫生之後，查不出病因，建議我做這個檢查。檢查的前一晚睡前，依醫囑吃了四顆藥，幫助排便，但是用藥須知卻忘了看，結果副作用就是半夜起來，肚子痛得要命，一連去了七、八回仍無法停歇，將近天明才又昏昏入睡。到了清早，仔細看了用藥須知，才知道服藥後，每隔20分鐘需飲水200cc，或許我就是吃了藥後直接上床去睡覺，沒看清楚這一條，讓我肚子白白痛了一晚上，真是又學了一課。
檢查當日，到了攝影室外，護士小姐吩咐先換上衣服，等候叫號。檢查時間約20分鐘到30分鐘，首先醫護人員會請你躺在攝影台上，吩咐你調整後位置後請勿亂動，接下來會先照射一張X光片，檢查是否已清腸完畢。五分鐘分後，另一名醫護人員會為你注射顯影劑，由右手上肢注射。隨著藥劑硫片全身，身體會開始發熱，喉嚨有灼熱感。經過五分鐘後，拍攝一張，又過十分鐘，又拍攝一張。此後，必須先到門外，進行喝水的步驟，要讓膀胱脹尿後再進行攝影。
在等候的同時，看著電視螢幕上撥放的衛教影片，一次又一次地看著，上面也清清楚楚的寫著吃下四顆藥丸後，必須每隔20分鐘喝下200cc的水，一共要喝3次。這又讓我回想到檢查前一天腹痛如絞的情形。到了下午醫生看診，竟被發現沒去驗尿，還被他凶了一頓，我只好傻笑以對。所幸檢查結果一切正常，真是感謝天父看顧，總之這趟醫院之行，又讓人體會身體保健的重要性。

詮釋與呈現

發表於 2011-07-16 更新於 2024-11-04

Interpretation 和 representation 這兩個字，近來一直在我腦海中縈繞著，由於需要撰寫一些有關這方面的文章，也上網搜尋了一下這面的資料。大致來說詮釋學是一門學問，然而當其應用在文化資產保存上的議題，就會出現不同的解讀方式。近來多元化或是多樣化的議題普遍受到重視，特別是在保護少數族群的做法與想法上，也以漸漸獲得大家的認同。 因此不同文化之間對於標的物所代表或醞含的意義，這個解釋權，就可以是很多元的。記得在電影航站情緣中，男主角因為國中內戰的關係，突然間變成沒有身分的國際游民，無法出境亦無法返國，就只能以機場為家。有一次有位入境的人帶著一些受管制的禁藥無法入境，然而這些藥品對他的家人非常的重要，不能被海關沒入，因此男主角急中生智上前與他用家鄉話對話時，故意捏造這是要給男子家中飼養的羊兒吃的，既是如此就可不管制。

從上述故事讀來就可以知道，原來物品的意義，其解釋權仍須在地化的考量，而非片面的斷章取義。走筆至止，又想到另一部電影，魔鬼代言人，片中男主角飾演一名大律師。在一次案件的委託之中，成功協助他的委託人脫罪，本來可能是傷害動物的一向罪則，但是大律師將傷害動物的這個舉動連結至宗教的信仰與活動，而成功說服法官。 總之，詮釋全是一個尊重少數族權且為了保存多元文化的做法，我想，世界萬物都是上帝所造，都是美好的，因此各民族之間能彼此尊重瞭解，也是好事一件。 延伸閱讀 http://life.fhl.net/Movies/netmovies/netm_01.htm

綠色校園新聞眉批

發表於 2008-04-28 更新於 2024-11-04 分類於 建築新聞 ， 綠色校園

中國時報2008.04.22
台北大學拔燈管年省電6萬度
陳至中／台北報導

校園推行節能省電，關冷氣、爬樓梯已經不夠看！台北大學規定住宿生必須負擔電費，學生為了省錢，索性提議減少宿舍的日光燈管，由學生團隊合作挑出不需要的燈管，一共拔掉八○六支，一年為學校省下六萬四千度電以及十五萬元電費。

台北大學住宿生負擔電費，指的是所有的電費還是部份的電費，是一般電費還是冷氣的電費，沒有交代清楚。

教育部廿一日正式啟動「綠色大學」校園節能減碳活動，並表揚表現優良的大學，其中遠東科技大學減少一三○萬度、台北大學減少八十萬度、朝陽科大減少七十五萬度，節能成效驚人。

不知道比較的基礎是以學年度還是黏度來做比較，有沒有考慮學生數逐年衰減的問題。減少的用電數是否與學校學生人數做出比較分析，還是只看總數而已

使用者付費住宿生負擔電費

宿舍是校內最耗電的地區之一，台北大學從九十七年開始實施「使用者付費」制度，住宿生除了管理費，每年還得另外掏出七百到八百元的電費。學生為了少花錢，要求校方拆掉多餘的日光燈管，經過學生自行評估後，走廊四支變兩支、太過密集的則整套拆除；宿舍雖然變暗了點，每年卻可節省六萬四千多度電。台北大學校長侯崇文表示，原本宿舍用電由學校統一支付，等於沒住宿的同學也要出錢，不符合使用者付費精神。校方和學生協商，決議建置計費管理系統，三五％的電費由學生負擔。

宿舍收費多收七八百元後，減少照明是否造成危險，校園死角？重要的冷氣是否有管制或是依用電度數而向使用者收費呢？原來的收費方式聽起來就怪怪的，如今負擔百分之三十五的參數如何計算出來呢？

政大推冷氣卡一千元三個月

「雖然要多付錢，學生卻沒什麼反抗，還主動要求節能，可見學生有足夠的環保素養，使用者付費政策功效顯著！」侯崇文說。

國內大學推行宿舍使用者付費所在多有，但大都只限於冷氣費，例如政大學生要享受冷氣，得先到福利社買「冷氣卡」，儲值一千元可供四人房用兩、三個月。

冷氣卡的推動使用，初設成本多少，預計多久回收，此外是否有溫度的限制，還是只有時間的限制？

台北大學三峽校區大都是新建大樓，方便統一規畫，才能設置計費系統，由學生分擔電費。除了計費設備，還設置了「電力監測系統」、「動力設備自動卸載技術」等。

宿舍用電之外，廣大校區的夜間照明也是一大開銷，台北大學規畫「安全走廊」，半夜其他地方盡量熄燈，既環保，又兼顧夜歸學生安全。

夜間走廊的構想相當不錯，值得推廣。

朝陽科大減碳綠覆蓋率九成

朝陽科大則把重點放在屋頂，除了太陽能光電和雨水回收系統，部分校舍還綠化成「空中花園」，阻隔熱氣進入校舍，改善高樓層的高溫現象。總務長張華南說，該校建物的「綠覆蓋率」達八九．六％，各項節能加總，每年約可節省七十五萬度、電費二百多萬元。

在「綠色校園宣示活動」記者會上，共表揚八所學校。中山大學利用數位電表及控制器監控用電，再也沒有「超約」罰款情形；暨大新建校舍嚴格依綠建築指標設計；大仁科大更換省電燈具，減緩用電量成長。其他獲表揚的學校還包括聖約翰科大、遠東科大、修平技院等。

其實總量管制的電量控制，真是讓人又期待又怕受傷害，因為總量管制下，可以訂出你預期的節省目標，不過頻繁的斷電次數，往往造成空調相關機器的不堪負荷，而直接減少機器可用壽命。此外，節約的指標不易取信於人。

附帶一提，數位電表已經壞了很久，聽說承包的軟體廠商有一些狀況，所以大家的數位看板都是漆黑一片的，已經好久了。

飲酒！飲酒！

發表於 2008-03-23 更新於 2024-11-04 分類於 joke

1978年，鄧小平復出，大局底定後，即出訪美國。首站到華盛頓，離開華府前一日舉辦感謝晚宴，宴會前讓新聞記者提問。他的翻譯人員首度出國開洋葷，大喝牛奶、大啃牛排，結果肚子鬧革命，一到會場就上洗手間蹲馬桶。

記者等得不耐煩，就開始發問。第一個記者問鄧小平：「請問美國的首都在哪一個城市？」鄧小平簡直鴨子聽雷，根本聽不懂，他猜想，中國人見到陌生人總是問：「請問貴姓？」他就硬者頭皮回答：「我姓鄧。」

美國記者聽到鄧小平準確回答，非常驚訝，第二個記者繼續追問：「你在華盛頓除了官式行程，還做了什麼活動？」鄧小平想，中國人問完「貴性」之後，通常會再問：「大名是什麼？」他就用四川國語回答：「小平。」

記者對他居然能用英語說shopping感到不可思議。第三個記者又考問他：「請問台灣的領導人是誰？」鄧小平眼見記者不懷好意，頻頻追問，開始焦急起來，他一面望著大門，看翻譯回來沒有，一方面對記者說：「你等會兒！」

記者以為他回答「李登輝」，佩服不已，又再問他：「請你研判，李登輝的接班人是誰？」鄧小平被問得不耐煩，揮揮手說：「隨便！水扁！」

美國記者以為他預言時為台北市議員的陳水扁將繼李登輝之後當總統，又繼續追問他：「陳水扁之後又是誰？」鄧小平實在招架不住了，為了阻止記者繼續發問，他舉起酒杯，高聲說：「飲酒！飲酒！」

面天山一日遊

發表於 2008-03-11 更新於 2024-11-04 分類於 生活雜記

周六才去騎了腳踏車，只租了兩台一大一小，我在後面緊追，果然運動量有夠大，周日一早勉強起身，八點時還沒意願，到了八點五分，心想睡覺也是一天，整裝就去走了面天池一趟。

搭捷運在北投站下車，接著要等小六公車，車次頗少，我就去研究了一下雙層腳踏車架，架上有車的都停滿了，剩下沒有車的都不能用，尚未修理完畢。不一會兒，車了來了，沿途經過北投市區，許多老先生老太太買完了菜趕搭公車回家，途經一段路，兩旁長滿了櫻花，遊人如織，花花綠綠，櫻花與遊客交織成一幅恬靜的圖畫，許多人在此下了車就去賞花了。車子蜿蜒而上終於到了海拔不知幾百公尺的清天宮，這是起站，準備要上路了，一開始一路上坡，頗感吃力，不過沿途見到許多登山客都能安步當車的上山，心想我也要趕緊練好身體，多爬一下山才好。

中間到了一個休息站，稍事休息一下後，吃了點水果就上路了，此後的路途相當好走，所以也看的許多人全家大小一起出動。中途有一個太子碑，可惜碑文有一點破壞，希望這類文物能夠被好好地保存。過不多久終於到了面天池，因為不是雨季所以沒有池水，不過卻另有一番風味，走到池中環顧四週，有點像神鬼戰士的場景，吃了自帶的午餐，休息了一下，只見雲瀑一波一波下來，煞是好看，由於天候不佳，展望不好，就沒繼續走上面天上，原路回程，反而感覺膝蓋不太舒服，因為連續下坡所以非常吃力，這次登山並沒有穿登山鞋及護膝所以算是輕裝上山，相機都沒有帶，照片下次再補了。

下山途中，看見許多身著道袍，手值令旗的宗教人士一行人約二十多人，正往山上去，原來是要去面天池朝拜的，因為剛剛在面天池休息時，就看到有人在為另一人運功灌頂了，這些人帶著令旗，鮮花，水果那還OK，不知道有沒有帶金紙就是。畢竟在山上焚燒金指，除了污染空氣外，也會帶來危險。

好容易到得山下，點了碗劍筍湯，臭豆腐吃了填一下肚子，吃完了車也到了，我提議要去北投吃一下蓬萊排骨酥，果然獲得老大首肯，提早在中和街下車，店裡人潮洶湧，點了一份半斤外帶，花費一百八十元，記得從前是一百五十元的，看了一下只有四支，拎著就往旁邊的公園前進，一人兩支相當過癮，只不過爬山運動量不大，吃的東西卻是熱量很大，期待下次的面天山之旅，或許可以順利登頂吧。
