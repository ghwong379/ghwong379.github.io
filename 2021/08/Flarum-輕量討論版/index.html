Flarum 輕量討論版

發表於 2021-08-03 更新於 2024-11-04

Flarum

討論版軟體百家爭鳴，功能各有優劣，從早期的開始使用phpBB2，接觸過的軟體不下10種，慢慢發現討論版軟體過於擁腫，執行速度緩慢，因此Flarum出世，也就打著輕量快速的特點。

安裝方式

Server Requirements
Before you install Flarum, it’s important to check that your server meets the requirements. To run Flarum, you will need:

  • Apache (with mod_rewrite enabled) or Nginx
  • PHP 7.3+ with the following extensions: curl, dom, gd, json,
  • mbstring, openssl, pdo_mysql, tokenizer, zip
  • MySQL 5.6+ or MariaDB 10.0.5+
  • SSH (command-line) access to run Composer

由於還在Beta版本，目前無法正確安裝，網上所提供的安裝方式為
composer create-project flarum/flarum

在執行這個指令前，先使用SSH登入，切換至web資料夾下後直行即可，之後輸入
flarum/public
